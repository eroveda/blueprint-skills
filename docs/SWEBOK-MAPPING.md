# SWEBOK Mapping

Blueprint Skills are aligned with IEEE SWEBOK v4.0 Knowledge Areas:

| Blueprint Category | SWEBOK Knowledge Area | What it Covers |
|---|---|---|
| DESIGN | Software Design (KA 4) | Data models, schemas, contracts, architecture |
| CONSTRUCTION | Software Construction (KA 5) | Coding, integration, business logic |
| SECURITY | Software Security (KA 14) | Auth, authorization, data protection |
| TESTING | Software Testing (KA 6) | Unit, integration, system, performance tests |
| OPERATIONS | SE Operations (KA 7) | Deployment, monitoring, configuration |

## Standards Stack

| Standard | Purpose | Where Applied |
|---|---|---|
| IEEE SWEBOK v4.0 (ISO/IEC TR 19759) | Knowledge taxonomy | Categorization of all skills |
| ISO/IEC/IEEE 12207 | Software lifecycle processes | Pipeline structure |
| ISO/IEC/IEEE 24748-5 | Software development planning | Outline generation |
| ISO/IEC/IEEE 16326 | Project management plan | Execution planning |
| ISO/IEC/IEEE 15289 | Documentation content | Documentation generation skills |

## How to Cite

Projects generated using Blueprint Skills can include in their README:

> Specifications generated following IEEE SWEBOK v4.0 taxonomy and ISO/IEC/IEEE 12207 software lifecycle processes.
